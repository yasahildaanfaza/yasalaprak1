SOAL TENTANG HTML

1. Apa yang kamu ketahui tentang HTML dan apa fungsi nya ?

HTML adalah kependekan dari Hypertext Markup Language yang merupakan sebuah bahasa markup. HTML adalah kode untuk membuat struktur halaman suatu website yang menarik, saling terhubung satu dengan yang lainnya, dan yang pasti dapat diakses melalui internet. Awalnya html ditemukan oleh Tim Berners-Lee pada tahun 1991. HTML adalah solusi untuk membantu ilmuwan dalam mengakses dokumen, namun kini html semakin berkembang pesat di dunia pemrograman web. HTML memiliki fungsi utama untuk membuat halaman website. HTML adalah pondasi utamanya, lalu dalam pembuatan website HTML biasanya berpasangan dengan CSS untuk mempercantik dan Javascript yang berguna untuk memberikan animasi dan kontrol fungsi.Dengan kode ini kamu bisa berbagi informasi melalui browserdi internet pada banyak orang. Namun, tentu saja kamu juga harus membeli layanan hosting agar halaman tersebut bisa diakses oleh banyak orang.

2. Jelaskan tentang tag dalam HTML !

Salah satu aspek penting dari HTML adalah penggunaan tag atau tanda, yang digunakan untuk menentukan struktur dan konten dari sebuah halaman web. Tag HTML merupakan sebuah tanda awal dan akhiran dari sebuah elemen. Tag dibuat dengan kurung <> dan berisikan nama tag dan terkadang ditambahkan dengan atribut misalnya seperti , , dan lainnya. Setiap tag HTML memiliki fungsi yang unik dan dapat memengaruhi cara sebuah halaman web ditampilkan dan berinteraksi dengan pengguna. Oleh karena itu, penting bagi para pengembang web untuk memahami fungsi-fungsi tag HTML ini agar mereka dapat membuat halaman web yang baik dan berfungsi dengan baik.

3. Sebutkan secara singkat aturan penulisan tag dalam HTML !

Saat menulis tag HTML, ada beberapa aturan dasar yang perlu diperhatikan. Berikut adalah tiga aturan dasar penulisan tag HTML:

  • Setiap Tag HTML Harus Memiliki Tanda Pembuka () dan Tanda Penutup (>) yang Sesuai.
  • Setiap Tag HTML Harus Ditulis dalam Huruf Kecil
  • Setiap tag HTML harus memiliki nama tag yang benar.

4. Apa yang dimkasud elemen dalam HTML ?

Elemen HTML adalah kode html yang ditulis mulai dari sebuah TAG pembuka sampai dengan TAG penutup dan diisi dengan konten di dalamnya.Perhatikan kode berikut:< tagname >content < tagname > disebut sebagai TAG pembuka, sedangkan disebut sebagai TAG penutup. Adapun sesuatu yang ada diantaranya "content" disebut konten. Nah secara keseluruhan kode itu mulai dari < tagname > sampai dengan termasuk konten di dalamnya itulah yang disebut dengan elemen HTML. Dalam HTML, tag, atribut, dan konten adalah elemen dasar yang digunakan untuk membuat dan mengatur struktur halaman web.

5. Jelaskan tentang tagname,attribute,dan content pada elemen!

a) Tag:Tag adalah elemen pembuka dan penutup dalam HTML yang digunakan untuk mendefinisikan struktur atau jenis konten. Tag selalu diawali dengan tanda kurung sudut ( <>) dan diakhiri dengan tanda kurung sudut tutup.

b) Atribut:Atribut adalah informasi tambahan yang ditambahkan ke dalam tag untuk memberikan karakteristik atau sifat tertentu pada elemen tersebut. Atribut selalu berada di dalam tag pembuka dan terdiri dari pasangan nama dan nilai (name="value").

c) Konten:Konten adalah teks atau elemen lain yang berada di dalam tag HTML dan terlihat di halaman web. Konten terletak di antara tag pembuka dan tag penutup.

6. Apakah elemen dapat ditumpuk?Apa yang terjadi bila demikian

Elemen HTML dapat ditumpuk atau dipesarangkan satu sama lain. Ini disebut elemen bersarang (nested elements). Artinya, satu elemen bisa berada di dalam elemen lain. Elemen HTML bersarang adalah struktur HTML di mana satu elemen ditempatkan di dalam elemen lain.Elemen yang melampirkan sering disebut sebagai "induk," sedangkan elemen yang dilampirkan disebut "anak." Elemen HTML Bersarang = Satu Elemen HTML Di Dalam Elemen HTML Lainnya. Ketika elemen bersarang, struktur hirarkis terbentuk. Elemen induk membungkus dan mencakup elemen anak. Elemen anak ini bisa juga memiliki elemen lainnya sebagai anak (cucu dari elemen induk). Kemudian Gaya (styles) dari elemen induk sering kali diturunkan ke elemen anak. Misalnya, jika elemen induk memiliki pengaturan warna teks tertentu, elemen anak akan mewarisi warna tersebut kecuali jika diatur berbeda secara khusus. Elemen anak dapat memiliki gaya, atribut, atau perilaku yang berbeda dari elemen induknya, meskipun mereka berada di dalam konteks elemen induk.

7. Jelaskan tentang Anchor pada HTML!

Anchor atau Tag jangkar adalah elemen HTML yang membuat tautan ke URL target. Jika diterapkan dengan benar, tautan dapat membungkus teks, gambar, atau sebagai tombol, sehingga pengguna dapat berinteraksi dengannya dan mengunjungi tujuan tautan. Tag jangkar memerlukan atribut href yang menentukan URL yang akan ditautkan. Atribut lain dapat membantu meningkatkan tampilan dan perilaku tautan, tetapi tidak wajib.

11. Apabila sebuah tag image tidak menampilkan gambar,apa penyebabnya ?

Penyebab tag image tidak menampilkan gambar yang pertama adalah kesalahan nama folder gambar di kode HTML. Kesalahan tersebut membuat server tidak akan bisa menemukan gambar yang diminta untuk ditampilkan.solusinya adalah mengecek nama folder gambar yang tidak muncul di kode HTML. Nama folder biasanya terletak di sebelah kiri nama file gambar. Peyebab selanjutnya adalah kesalahan penulisan nama file gambar. Ketika mengecek nama folder di kode HTML, pastikan juga nama file gambar sudah benar. Kalau ada pengejaan yang salah, akibatnya gambar tidak muncul di halaman web. Selanjutnya kesalahan penulisan ekstensi file gambar Penulisan ekstensi file yang salah di kode HTML juga bisa membuat gambar tidak muncul. Contohnya, gambar yang dipasang di halaman web berjenis PNG, tetapi di kode tertulis “.jpg”.Selain itu, perhatikan juga penggunaan huruf kapital dan kecil di nama file sebelum diunggah ke website. Sebab, gambar tetap tidak bisa ditampilkan jika Anda mengetikkan “.jpg”, sedangkan yang tertulis di nama file-nya diikuti “.JPG”.Penyebab selanjutnya gambar tag image tidak menampilkan gambar mungkin file gambar yang digunakan error

13. Apa yang terjadi apabila atribut autoplay pada tag video tidak berjalan dan bagiamana caranya agar autoplay tersebut dapat berfungsi

Jika atribut autoplay pada tag < video > tidak berjalan, ada beberapa alasan yang mungkin menyebabkan hal tersebut. Kebijakan Browser terkait autoplay merupakan salah satu penyebabnya . Banyak browser modern (seperti Chrome, Firefox, dan Safari) memblokir pemutaran video otomatis secara default, terutama jika video memiliki suara. Ini dilakukan untuk meningkatkan pengalaman pengguna dan mencegah konten yang mengganggu. Penyebab selanjutnya adlah video menagandung audio , Browser biasanya hanya mengizinkan pemutaran otomatis jika video tidak mengandung audio atau dalam keadaan mute. Jika video memiliki suara dan atribut muted tidak ditambahkan, autoplay kemungkinan tidak akan berjalan. Kemudian beberapa pengguna ada yang menonaktifkan fitur autoplay di pengaturan browser mereka, atau ada ekstensi yang memblokir pemutaran otomatis yang juga menyebabkan autoplay pada tag < video > tidak berjalan.
